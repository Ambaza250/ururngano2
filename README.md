
# Urungano Platform

**Urungano** — A safe, trusted, and youth-friendly digital platform providing information and support on **Sexual & Reproductive Health (SRH)** and **Mental Health** in **Kinyarwanda**.

![Urungano Logo](urungano-logo.jpeg)

## 🌟 About the Project

Urungano is a multi-page static website designed to break the stigma around mental health and sexual/reproductive health in Rwanda. It offers reliable, accessible, and confidential information, an AI-powered assistant, emergency helplines, and a location-based service finder — all in **Kinyarwanda**.

The platform aims to make health education inclusive, especially for young people and persons with disabilities.

---

## ✨ Key Features

- **Bilingual-friendly** (Primarily Kinyarwanda)
- **Urungano AI Chat Assistant** – Confidential AI support
- **Mental Health Section** with resources and information
- **Sexual & Reproductive Health (SRH)** resources
- **Emergency Helplines** (24/7):
  - Mental Health: **114**
  - Isange One Stop Center: **3550**
  - Police/Emergency: **112**
- **Nearby Service Finder** (Police, Hospitals, Isange Centers) with map
- **Responsive design** – Works well on mobile and desktop
- **Voice input support** in chat (planned/enhanced)

## 📱 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Ambaza250/ururngano2.git
   cd ururngano2
   ```

2. Open any `.html` file in your browser (e.g. `index.html`)

No build step required — it's a static website.
